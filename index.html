Introduction

Thanks for checking out Baker Street. In the time since we've released Baker Street, some amazing new technologies have been released. In particular, Lyft has released Envoy, a modern L7 proxy and load balancer. If you're interested in tackling the problems that Baker Street addresses, we've published a few articles about using Envoy as part of the Microservices Architecture Guide:
  • Getting started with Lyft Envoy for microservices resilience
  • Deploying Envoy with a Python Flask webapp and Kubernetes
You may also want to check out a few of our other open source projects for building productive development workflows on Kubernetes:
  • Telepresence lets you develop and debug Kubernetes services locally
  • Forge lets you get source running on Kubernetes quickly and easily
  • Ambassador is a Kubernetes-native API Gateway built on Envoy
